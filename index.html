Is Pericos Crowded?

This site determines how crowded Taqueria Los Pericos is, so you're never surprised by a line.

How It Works
This site, using anonymous GPS data from smartphones, estimates the number of people that are currently inside of Los Pericos and weighs this against historical averages. It then boils this information down into a simple message about the crowd situation at Santa Cruz's favorite taqueria.
